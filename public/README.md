# Public Assets Directory

Place your images, logos, and other static assets in this directory.

## Suggested locations:

- **Logo**: `public/logo.png` or `public/logo.svg` (for the main app logo)
- **Favicon**: `public/favicon.ico` (for browser tab icon)
- **Icons**: `public/icons/` (for various icons)
- **Images**: `public/images/` (for any background images or illustrations)

## Usage in code:

To reference files in this directory, use paths starting with `/`:

```tsx
// Example: Using a logo
<img src="/logo.png" alt="Logo" />

// Example: Using an image
<img src="/images/background.jpg" alt="Background" />
```
